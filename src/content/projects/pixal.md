![PIXAL](/images/pixal.png)

PIXAL adalah aplikasi web seleksi (culling) foto tercepat yang dirancang untuk mempercepat workflow fotografer profesional. Menggunakan teknologi Local-First, PIXAL memungkinkan user memilah ribuan foto RAW & JPG langsung melalui browser tanpa perlu upload ke internet dan tanpa install software berat. Cukup login, gunakan keyboard shortcut untuk seleksi kilat, dan biarkan sistem memindahkan file ke folder kategori secara otomatis di komputer user. Solusi hemat waktu, hemat kuota, dengan privasi data yang 100% aman.


Aplikasi ini saya buat dalam rangka mengeksplorasi teknologi File System API dan bagaimana mengimplementasikan workflow fotografer profesional di web. Karena selama ini, aplikasi sortir foto identik dengan software berat yang harus diinstall di komputer. Padahal, dengan teknologi web yang semakin maju, kita bisa membuat aplikasi sortir foto yang ringan dan mudah diakses. Selain itu, saya juga ingin membuktikan bahwa aplikasi web bisa bersaing dengan aplikasi desktop dalam hal performa dan fitur. 

Dalam membangun aplikasi ini, saya menggunakan teknologi-teknologi berikut:
- Node.js
- PostgreSQL
- React
- TypeScript
- ShadcnUI
- File System API
- Midtrans as payment gateway
